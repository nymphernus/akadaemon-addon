# 🌀Akadaemon Addon

![Minecraft 1.7.10](https://img.shields.io/badge/Minecraft-1.7.10-blue?style=for-the-badge)
![Forge](https://img.shields.io/badge/Forge-Required-orange?style=for-the-badge)

**Akadaemon Addon** — это техномагический аддон для Minecraft 1.7.10, выступающий связующим звеном между индустриальными и магическими модификациями.

---

## ⚙️Ключевые системы
### ⚡Техномагическая архитектура
Аддон внедряет систему кросс-модовой конвертации ресурсов. Механизмы мода позволяют использовать электрическую мощность (EU) для манипуляций с магическими аспектами и наоборот.

### ⚡Металлургия и Сплавы (Tinkers' Construct)
Расширена функциональность плавильни. Новые жидкости и их смеси.

### ⚡Экосистема Baubles и Снаряжения
Пост-квантовый сет брони через наполнение на алтаре и новые артефакты, дающие пассивные улучшения.

### ⚡Лор и Исследования
Весь контент интегрирован в Таумономикон через новую вкладку.

### ⚡Технические особенности
Поддержка `OreDictionar` для полной совместимости с любыми сторонними сборками и возможность настройки имен ресурсов и баланса предметов через конфиг.

---

## ⚙️Зависимости
Для работы мода **обязательно** наличие следующих модификаций:

| Мод | Рекомендуемая версия |
| :--- | :--- |
| **IndustrialCraft 2 Exp** | `2.2.827` |
| **Thaumcraft** | `4.2.3.5` |
| **Tinkers' Construct** | `1.8.8` |
| **Baubles** | `1.0.1.10` |

---

## ⚙️Установка
1. Скачайте актуальную версию `.jar` файла из раздела [Releases](../../releases).
2. Поместите файл в папку `/mods/` вашего игрового клиента или сервера.
3. При первом запуске будет сгенерирован файл конфигурации `config/akadaemon.cfg`
